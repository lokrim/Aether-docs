# Aether Documentation

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## How to run

Developer can run the documentation website using this command:

```bash
# install dependencies
npm install

# run the application
npm start
```

## Documenting

### How to add documentation

Write the documentation as markdown file (.md) in `/docs` folder and update it in `sidebars.js`.

### 한국어 다큐멘팅 방법

`/docs` 폴더 안에 있는 구조를 그대로 가져오되 `.md` 파일을 `/aether-docs/i18n/ko/docusaurus-plugin-content-docs/current`에 넣으시면 한국어 버젼도 보입니다.
